# EDA Template

This repository contains a reusable Exploratory Data Analysis (EDA) notebook template.

## Purpose
To standardize the EDA process across projects and ensure consistent analysis structure.

## Sections Included
- Imports
- Data Loading
- Data Overview
- Missing Value Analysis
- Univariate Analysis
- Bivariate Analysis
- Feature Engineering Ideas
- Business Insight Summary

## Usage
Copy the notebook into a new project and modify according to the dataset.
